# Инженерный калькулятор УЭЦН

**12 расчётных модулей для технолога нефтедобычи.**  
Открываешь, вводишь цифры, получаешь результат. Работает с телефона в поле.

[Открыть калькулятор](https://technologistNT.github.io/engineering-calculator/)

---

## Зачем это нужно

Каждый технолог УЭЦН знает: чтобы посчитать отпайку ТМПН, запас напора или режим АПВ, приходится лезть в кучу экселевских таблиц. Где-то не сходится, где-то формула сломана, где-то коллега случайно удалил ячейку. Надоело.

Я собрал всё в одном месте. Без воды, без модных корреляций — только проверенные методики и реальный опыт эксплуатации.

---

## Что внутри

| Модуль | Что считает |
|--------|-------------|
| **АПВ / ПКВ** | Время работы и простоя при циклической откачке |
| **Кпр** | Коэффициент продуктивности (4 варианта расчёта) |
| **Давление на приёме** | Пересчёт динамического уровня в давление |
| **Давления и уровень** | Сходимость замеров уровня и трубного давления |
| **Кабель / ПЭД** | Потери напряжения, подбор отпайки ТМПН, загрузка двигателя |
| **Напор и резерв** | Хватает ли напора насоса |
| **Частотное регулирование** | Изменение подачи, напора, мощности при смене частоты |
| **Свободный газ (β)** | Объёмная доля газа на приёме насоса |
| **Объёмы скважины** | Затруб, НКТ, ёмкости, масса нефти |
| **Охлаждение ПЭД** | Минимально допустимый дебит для охлаждения двигателя |
| **Штуцер** | Подбор диаметра штуцера |
| **Глушение** | Объёмы и давление при глушении скважины |

В каждом модуле — встроенная подсказка с простым примером. Не надо вспоминать формулы.

---

## Как пользоваться

1. Открываешь ссылку в браузере — на компьютере или телефоне.
2. Вводишь цифры — результат считается сразу.
3. Нужно в отчёт — выгружаешь в PDF или Excel одной кнопкой.

Никакой установки. Никаких серверов. Просто работает.

---

## Почему не Excel

- Логика расчёта внутри — формулу случайно не удалишь.
- Единый вид на всех устройствах — и на компе, и на телефоне.
- Цветовая индикация: зелёный — норма, оранжевый — внимание, красный — риск.
- Все параметры сохраняются в браузере — открыл, а цифры с прошлого раза уже на месте.
